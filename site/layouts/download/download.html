Description

  • Interaction tables are comma separated tables (csv) of transcription factor - target gene interactions that contain either interactions validated by small-scale experiments or large-scale experiments or these two data altogether. All tables contain the following data: Uniprot IDs, protein names, NCBI Gene IDs of the transcription factors and the target genes; the names of the detection methods; the Pubmed IDs of the original publications; the name of the organism; the source databases; and indication about if the data were confirmed by small-scale evidences.
  • interaction GMTs contain transcription factor - target gene interactions in GMT (Gene Matrix Transposed) format that suitable for gene set enrichment and overrepresentation analysis. The detailed description of the format is available in the FAQ.
  • Interaction MITABs contain transcription factor - target gene interactions in HUPO-PSI MITAB 2.8 format. The detailed description of the format and a header for MITAB tables are available in the FAQ.
  • Binding site tables are comma separated tables (csv) of binding site annotations that contain the unique internal TFLink IDs of binding sites; Uniprot IDs and protein names of the transcription factors; the names of the detection method; Pubmed IDs of the original publications; the organism and the source database; the version of the genome assembly; the name of the chromosome; the start and end coordinates of site; and the strand.
  • Binding site sequence files are fasta files containing the DNA sequences of the transcription factor binding sites. The header of each sequence contains the unique internal TFLink ID of the binding site; the Uniprot ID and protein name of the transcription factor; the version of the genome assembly; the name of the chromosome; and the the start and end coordinates of site.
