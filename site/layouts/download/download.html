Description

  • Interaction table: comma separated table of transcription factor - binding site interactions.
  • Interaction MITAB: interaction information in the HUPO-PSI MITAB 2.8 format. Format description and a header for MITAB tables can be found in the FAQ.
  • Binding site table: comma delimited table of binding site annotations. Contains the TFLink IDs of binding sites, transcription factor IDs and names, names of source databases, detection methods, PubMed accession numbers of publications the binding sites were described in, genome assembly and genomic locations.
  • Binding site sequences: FASTA files containing the DNA sequences of the transcription factor binding sites. The header of each sequence contains the internal TFLink ID of the side, the Uniprot accession and name of the transcription factor, the name of the genome assembly and the genomic coordinates of the binding site (chromosome:start-end) separated by semicolons.
