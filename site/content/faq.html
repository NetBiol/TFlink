What does a protein page contain?

When you select a transcription factor or a target gene protein from the Browsing table a protein page will be opened.

What does a transcription factor protein page contain?

Besides basic information and a link to the Uniprot page of the protein there is a table that contains information about the target genes of the transcription factor.
If there are known binding sites of the transcription factor you will find a table containing their genomic locations and a list containing their DNA sequences in fasta format below the table.

What does a target gene protein page contain?

Besides basic information and a link to the Uniprot page of the protein there is a table that contains information about the transcription factors of the target gene.

What does a transcription factor and target gene protein page contain?

Some proteins act like transcription factors and target genes simultaneously. In some cases a transcription factor can regulate its own gene too.
Besides basic information and a link to the Uniprot page of the protein there is a table that contains information about the target genes of the protein.
There is also a table that contains information about the transcription factors of the protein.
If there are known binding sites of the transcription factor you will find a table containing their genomic locations and a list containing their DNA sequences in fasta format below the table.

What are the downloadable data formats?

What do interaction tables contain?

Interaction tables are comma separated tables (csv) of transcription factor - target gene interactions that contain either interactions validated by small-scale experiments (small-scale interaction table) or large-scale experiments (large-scale interaction table), along with the following data: Uniprot IDs, protein names, NCBI Gene IDs of the transcription factors and the target genes; the names of the detection methods; the Pubmed IDs of the original publications; the name of the organism and the source database.

What do MITAB tables contain?

Interaction MITABs contain transcription factor - target gene interactions in HUPO-PSI MITAB 2.8 format.

What is the MITAB format?

MITAB 2.8 is a standardized format introduced by the Human Proteome Organization (HUPO) which captures binary molecular interactions using standardized content and vocabulary. The MITAB files are tab-delimited tables containing 46 columns. A header for the MITAB tables is available here.
For more information on the HUPO-PSI's molecular interaction format see: link
Sivade Dumousseau M et al. (2019) Encompassing new use cases - level 3.0 of the HUPO-PSI format for molecular interactions. BMC Bioinformatics 19(1):134. doi: 10.1186/s12859-018-2118-1.

What do binding site tables contain?

Binding site tables are comma separated tables (csv) of binding site annotations that contain the unique internal TFLink IDs of binding sites; Uniprot IDs and protein names of the transcription factors; the names of the detection method; Pubmed IDs of the original publications; the organism and the source database; the version of the genome assembly; the name of the chromosome; the start and end coordinates of site; and the strand.

What do binding site sequence files contain?

Binding site sequence files are fasta files containing the DNA sequences of the transcription factor binding sites. The header of each sequence contains the unique internal TFLink ID of the binding site; the Uniprot ID and protein name of the transcription factor, the version of the genome assembly; the name of the chromosome; and the the start and end coordinates of site.

Where can I find the the interaction and binding site data?

Where can I find the interactions and binding sites based on small-scale experiments?

For data of small-scale experiments we provide options for browsing, searching and downloading data in different standardized formats (MITAB, csv, gmt and fasta files that include Uniprot IDs, protein names, PubMed IDs, etc.).

Where can I find the interactions and binding sites based on large-scale experiments?

Because of their quantity, the interactions and binding sites based on large-scale experiments are only available in forms of downloadable files, therefore cannot be browsed on the TFLink website. The downloadable files are accessible in standard formats: MITAB, csv, gmt and fasta files that include Uniprot IDs, protein names, PubMed IDs, etc.

Where can I find the data of Danio rerio?

From Danio rerio only large-scale data was available in the source databases, therefore you can find these interaction and binding site data under the Downloads menu.

Is TFLink freely available?

TFLink is freely available for non-commercial use.
